# PyCubes
Небольшой майнкрафт-клон написанный на python.

Добавлены различные блоки  - для творчества.

Управление:
"-" не ставит блок
"0" белый блок
"1" блок травы
"2" блок камня
"3" блок кирпича 
"4" блок земли
"5" стекло
"6" магнитный блок
правая кнопка мыши - ставит блок
левая кнопка мыши  - удаляет блок
"g" - отключить гравитацию
"h" - включить гравитацию
